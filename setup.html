üöÄ Quick Start

Step 1: Start a Local Server

Choose one of these methods:

Python 3:

cd web
python -m http.server 8000

Node.js (http-server):

cd web
npx http-server

PHP:

cd web
php -S localhost:8000

Step 2: Open the App

Navigate to: http://localhost:8000

Step 3: Connect Arduino

  1. Power on your Arduino Nano 33 IoT
  2. Click "Connect Arduino" in the web app
  3. Select your device from the browser's Bluetooth dialog
  4. Choose a characteristic with NOTIFY capability
  5. Start playing!

‚öôÔ∏è Arduino Setup

Your Arduino should broadcast IMU data via BLE in this format:

ax,ay,az,gx,gy,gz\n

Example Arduino sketch structure:

#include <ArduinoBLE.h>
#include <Arduino_LSM6DS3.h>

BLEService imuService("19B10000-E8F2-537E-4F6C-D104768A1214");
BLECharacteristic imuChar("19B10001-E8F2-537E-4F6C-D104768A1214",
                          BLERead | BLENotify, 512);

void setup() {
  BLE.begin();
  IMU.begin();
  BLE.setLocalName("Violin IMU");
  BLE.setAdvertisedService(imuService);
  imuService.addCharacteristic(imuChar);
  BLE.addService(imuService);
  BLE.advertise();
}

void loop() {
  float ax, ay, az, gx, gy, gz;
  if (IMU.accelerationAvailable()) {
    IMU.readAcceleration(ax, ay, az);
    IMU.readGyroscope(gx, gy, gz);

    String data = String(ax) + "," + String(ay) + "," +
                  String(az) + "," + String(gx) + "," +
                  String(gy) + "," + String(gz) + "\n";

    imuChar.writeValue(data.c_str());
  }
  delay(100);
}

üîß Configuration

Motion Sensitivity

Adjust these values in app.js:

this.MAX_MOTION_SPEED = 250.0;  // Max motion reference
this.motionThreshold = 0.15;    // 15% default threshold
this.maxHistorySize = 5;        // Smoothing window

Volume Gradient

The app uses an ease-in-out curve for natural volume transitions:

  • Above threshold: Full volume
  • Below threshold: Smooth gradient
  • Very low motion: Automatic pause

üêõ Troubleshooting

Can't Connect to Device

  • Ensure Bluetooth is enabled on your computer
  • Verify the Arduino is powered and advertising
  • Try refreshing the page and reconnecting
  • Check browser console (F12) for error messages

No Audio Playing

  • Check that IMU Playback toggle is ON (green)
  • Verify motion speed exceeds the threshold
  • Try Test Playback to verify audio file loads
  • Check browser audio permissions

Jerky Motion Detection

  • Increase motion smoothing: this.maxHistorySize = 10
  • Adjust threshold to higher value for stability
  • Ensure stable Bluetooth connection
